# Hubble Parameter Estimation

This project calculates the Hubble parameter using cosmological data to study the expansion of the universe.

## Description
The notebook demonstrates numerical computation of the Hubble parameter using observational data and Python-based scientific libraries.

## Technologies Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## Learning Outcome
- Implemented cosmological equations in Python
- Performed numerical analysis of astronomical data
